Endpoint για την καταχώρηση της επιλογής απάντησης με αναγνωριστικό optionID κατά το γεγονός απάντησης με αναγνωριστικό session στην ερώτηση questionID για το ερωτηματολόγιο questionnaireID.

Resource URL
http://localhost:9103/intelliq_api/doanswer/{questionnaireIID/{questionID}/{session}/{optionID}
Parameters
questionnaireID (Query)
required

Το αναγνωριστικό του ερωτηματολογίου.

Type: int
questionID (Query)
required

Το αναγνωριστικό της ερώτησης.

Type: int
session (Query)
required
Τέσσερις τυχαίοι λατινικοί χαρακτήρες που αναγνωρίζουν μοναδικά ένα γεγονός απάντησης.
Type: char
Example Values: XVBI
optionID (Query)
optional
Type: int
Curl Example
$ curl -X POST -d '' "http://localhost:9103/intelliq_api/doanswer/{questionnaireIID/{questionID}/{session}/{optionID}?questionnaireID=&questionID=&session=XVBI&optionID="
